## Why it is worth using on free tier

GitHub Pages is a practical default when your project can be fully static. It removes backend complexity and pairs naturally with repository workflows.

## Operational notes

- Keep output paths predictable for static hosting constraints.
- Use repository-level environment variables to handle base path values.
- Validate exported links after each deployment.
