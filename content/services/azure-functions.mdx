## Why it is worth using on free tier

Azure Functions is one of the strongest free-tier options for teams that need API endpoints and event processing without paying for always-on infrastructure. The model works well when traffic is bursty and predictable.

## Operational notes

- Keep function app startup weight low to reduce cold starts.
- Put strict retry and dead-letter behavior around queue-triggered functions.
- Pair it with Azure Storage and static frontends for low-cost full-stack projects.
